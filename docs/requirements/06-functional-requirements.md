# Functional Requirements

1. The system shall allow users to register and manage personal accounts.
2. The system shall allow users to configure trusted contacts.
3. The system shall provide a quick SOS activation mechanism for emergency situations.
4. The system shall capture and transmit the userâ€™s real-time location during an emergency.
5. The system shall notify trusted contacts when an SOS alert is triggered.
6. The system shall initiate a countdown timer after an SOS alert is activated.
7. The system shall automatically escalate the alert if the user does not respond within the defined time period.
8. The system shall allow users to cancel an active SOS alert.
9. The system shall record and store emergency alert events securely.
10. The system shall allow trusted contacts to acknowledge received alerts.
