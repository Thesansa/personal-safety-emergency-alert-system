# Use Cases

## UC-01: Register / Login
Primary Actor: User  
Description: Allows a user to access the system securely.

## UC-02: Manage Trusted Contacts
Primary Actor: User  
Description: Allows the user to add, edit, or remove trusted contacts.

## UC-03: Trigger SOS Alert
Primary Actor: User  
Secondary Actor: System  
Description: Allows the user to activate an emergency alert.

## UC-04: Cancel SOS Alert
Primary Actor: User  
Description: Allows the user to cancel an active SOS alert.

## UC-05: View SOS Status
Primary Actor: User  
Description: Allows the user to view the current state of an SOS alert.

## UC-06: Receive Emergency Alert
Primary Actor: Trusted Contact  
Description: Notifies trusted contacts when an SOS is triggered.

## UC-07: View User Location
Primary Actor: Trusted Contact  
Description: Allows trusted contacts to view the user's live location during an active SOS.

## UC-08: Acknowledge Alert
Primary Actor: Trusted Contact  
Description: Allows trusted contacts to acknowledge the emergency alert.

## UC-09: Resolve SOS Alert
Primary Actor: Trusted Contact  
Description: Allows confirmation that the user is safe.

## UC-10: Escalate SOS Alert
Primary Actor: System  
Description: Automatically escalates the alert if the user is unresponsive.
