# User Stories

## User (Person in Danger)

As a User, I should be able to:
- Trigger an SOS alert quickly to notify trusted contacts in an emergency
- Remain logged into the system to allow rapid SOS activation
- Cancel an active SOS alert to prevent false or accidental notifications
- Share my real-time location during an active SOS alert
- Add and manage multiple trusted contacts
- View the current status of an SOS alert
- Confirm safety and resolve the alert once the emergency is over
